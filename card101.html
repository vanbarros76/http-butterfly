HTTP Butterfly

101 Switching Protocols

O código de resposta HTTP 101 - Switching Protocols, indica um protocolo para o qual o servidor alterna. O protocolo é especificado no cabeçalho de solicitação Upgrade recebido de um cliente. O servidor inclui nesta resposta um cabeçalho de resposta de atualização para indicar o protocolo para o qual ele mudou.
